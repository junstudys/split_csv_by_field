# CSV 智能拆分工具 - GUI 设计快速参考

> **完整设计文档**: 请参阅 `04-GUI界面设计文档.md`

---

## 🎨 设计亮点

### 核心设计原则

1. **零学习成本** - 3分钟内完成首次拆分
2. **现代化UI** - 参考VS Code/Notion/Linear风格
3. **高效反馈** - 实时进度显示，友好的错误提示
4. **深浅主题** - 支持浅色/深色/自动主题切换

---

## 🎯 主流程

```
首页 → 文件选择 → 字段配置 → 拆分设置 → 预览确认 → 执行拆分 → 结果展示
```

**步骤导航器**：顶部显示当前步骤（1/4, 2/4...）

## 💡 关键交互

### 拖拽排序

- 字段配置页支持拖拽调整拆分顺序
- 文件可从文件管理器拖放到窗口

### 实时预览

- 配置更改时实时更新输出文件名预览
- 显示预计输出文件数

### 进度反馈

- 总体进度条 + 当前文件进度条
- 已完成文件列表
- 实时统计信息（行数、速度、剩余时间）
